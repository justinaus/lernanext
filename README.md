# Lernanext

http://lernanext.vercel.app/

## Install and Start (PC)

```sh
cd packages/pc

yarn install

yarn start
```

```
├── packages
│   ├── shared
│   │   ├── hooks
│   │   ├── interfaces
│   │   └── utils
│   ├── pc
│   └── mo
```

- lerna

### shared
